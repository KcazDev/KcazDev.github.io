IPG Property Manager App

A redesigned mobile app experience tailored specifically for IPG, a property management company. About the Project

The goal of this project was to redesign an existing project management app with a specific client in mind — IPG, a property management company. Rather than designing a generic tool, every screen and feature was tailored to how IPG's managers and maintenance teams actually work. The app lets staff log new property projects, fill out detailed job information, view current projects, and submit work orders for manager approval.

The design process started with lo-fi wireframes to map out the user flow and layout before moving into high-fidelity completed screens. This two-stage approach meant the structure and navigation were fully thought through before any visual design decisions were made, resulting in a cleaner and more intentional final product.
