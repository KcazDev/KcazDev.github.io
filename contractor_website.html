Contractor AI Website

A full-stack construction estimator web app that uses AI to generate cost estimates based on project dimensions and materials. Users input their project specs — length, width, height, material type, and load type — and the app validates the data and produces an AI-powered cost breakdown. Built with HTML, CSS, JavaScript, Bootstrap, and a Node.js backend.

HTML CSS JavaScript Node.js Bootstrap AI Integration

About the Project

This project was built as a practical full-stack application that solves a real problem — getting quick, accurate construction cost estimates. The user fills out a project input form with dimensions (length, width, height), material type (wood, steel, or concrete), load type (static or dynamic), and cost per unit. The app validates every field with detailed error messages before passing the data to the AI to generate an estimate.

The form includes robust client-side validation — every field is checked for missing values, negative numbers, and unrealistic inputs like a 1000ft wall. If anything fails, an error banner lists every issue and the form stops submission until they're fixed. Once validated, the data is passed via sessionStorage to a results page where the AI-generated estimate is displayed.

Technical Details

The frontend was built with Bootstrap 5 for responsive layout and form components, with custom CSS layered on top for styling. JavaScript handles all form validation, error state management, and sessionStorage for passing data between pages without a page reload. The backend uses Node.js to handle the AI API calls and return the generated estimates to the client.

Key features include real-time field validation that clears errors as the user types, a scrolling error banner that summarises all issues at once, a loading screen while the estimate generates, and a clean results page displaying the full cost breakdown.
